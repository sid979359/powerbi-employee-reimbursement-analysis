## What this project covers

- Cleaned and standardized expense types and project names using Power Query
- Handled missing currency values with conditional logic
- Converted multi-currency amounts into INR
- Built DAX measures for:
  - Total reimbursed amount
  - Project-wise reimbursement
  - Declined requests count
- Designed an interactive Power BI dashboard with slicers and charts
